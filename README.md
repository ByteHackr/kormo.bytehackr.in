# KormoNama

Free & open source resume builder. No signup, no fees, 100% private.

**Live:** [kormo.bytehackr.in](https://kormo.bytehackr.in)

## Features

- Live preview
- 14+ templates
- PDF export
- Auto-save (localStorage)
- Dark/Light theme

## Run Locally

```bash
python3 -m http.server 8000
```

Open http://localhost:8000
